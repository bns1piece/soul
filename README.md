# blade: Back-end Project

넷마블 모바일게임 **블레이드&소울 레볼루션**에서 필드 보스의 등장 시간을 **집단지성**을 이용하여 예측하고 알려주는 보조도구.

## 기능

### 리젠시간에 따라서 직전 리젠시간, 잡은시간 및 예상 등장시간 저장
- 채널별 구분
- 제룡림 10분, 대사막 60분, 수월평원 120분, 백청산맥 180분
- 이전 24시간 히스토리 저장
